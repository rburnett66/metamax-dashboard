# Project Requirements — Design Stage (Habit Tracker)

## Design Goals
- Calm, minimalist, and encouraging visual language
- Instant visual feedback on progress (streaks, completion rings)
- Delightful micro-interactions that feel rewarding
- Excellent readability in both light and dark environments

## Key Screens to Design
1. Onboarding (3 screens)
2. Home / Dashboard (habit overview + today’s progress)
3. Habit creation / editing flow
4. Habit detail view with history
5. Settings & reminders

## Design System Requirements
- Must use Gulf Oil palette (cyan #00AEEF + orange #FF6B00)
- 4px spacing system
- Rounded-2xl (16px) for cards
- Inter font family
- Strong focus states and accessibility

## Success Criteria
- Prototype tested with 5 real users → 4.5+ / 5 ease-of-use score
- All screens pass WCAG 2.1 AA contrast
- Design system documented and handed off cleanly
